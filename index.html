Sistearth-frontend

Frontend with Ember for Sistearth v4

Tools

  • npm install -g ember-cli
  • npm install -g bower
  • npm install -g phantomjs
  • npm install --save-dev broccoli-sass

Installation

  • git clone this repository
  • npm install
  • bower install

Running

  • ember server
  • Visit your app at http://localhost:4200.

Running Tests

  • ember test
  • ember test --server

Building

  • ember build
